## Hardware Info

There are several ways.

- `lshw` command.
- `cat /proc/cpuinfo`.
- `neofetch` command available via `sudo apt install neofetch`.

# RAM

Use the `free` command. You can also consult the content of the `/proc/meminfo` file with `cat /proc/meminfo`.
